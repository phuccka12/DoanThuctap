🎯 HOW IT WORKS:

Audio File
    ↓
[Whisper STT] → Transcript
    ↓
[extract_acoustic_features] → 44 acoustic features
    ├→ 13 MFCCs (âm sắc)
    ├→ 13 Delta MFCCs (biến thiên)
    ├→ 13 Delta-Delta MFCCs (gia tốc)
    └→ 5 physics: pitch, jitter, shimmer, energy, silence
    ↓
[XGBoost Model] → Score + Confidence
    ↓
[Gemini API] → Detailed feedback + tips
    ↓
[format_speaking_response] → JSON for frontend
    ↓
Response (70% Gemini + 30% XGBoost = Final Score)

═══════════════════════════════════════════════════════════════

📊 API ENDPOINT:

POST /api/speaking/evaluate-hybrid

Request:
  - audio (file) - WAV, MP3, WEBM, OGG
  - transcript (text) - Transcribed speech
  - question (text) - The speaking prompt

Response:
{
  "success": true,
  "source": "hybrid-xgboost-gemini",
  "scores": {
    "overall": 7.5,           ← Combined score (70% Gemini + 30% XGBoost)
    "pronunciation": 7.8,     ← From Gemini
    "fluency": 7.2,           ← From Gemini
    "confidence": 7.4         ← From Gemini
  },
  "band": "7.0-band",         ← IELTS band
  "strengths": [...],         ← What's good
  "areas_for_improvement": [...],  ← What to fix
  "feedback": "...",          ← Vietnamese feedback
  "tips": [...],              ← Actionable tips
  "xgboost_acoustic": 7.65,   ← Your model score
  "xgboost_confidence": 0.87  ← Model confidence (0-1)
}
